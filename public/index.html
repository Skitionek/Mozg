Welcome to Mozg

A federated GraphQL query layer — one endpoint for every data source

⚠️ Experimental & work in progress — not recommended for production use

⚡
Universal endpointQuery PostgreSQL, MySQL, Neo4j, MongoDB, REST APIs and more through a single /graphql URL — no ETL, no data duplication
🔗
Cross-entity relationsTraverse hasMany, hasOne and belongsTo relationships across tables and REST endpoints in a single query
🔍
Schema introspectionDiscover tables, columns and types in any live database dynamically — no prior configuration required
🦉
OWL ontology supportParse OWL ontologies (Turtle, RDF/XML, Manchester) and generate GraphQL types from their TBox automatically
